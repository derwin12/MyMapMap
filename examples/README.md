# MyMapMap Example Projects

## demo.mmp

A starter project demonstrating the three main layer types:

| Layer | Source | Shape | Notes |
|---|---|---|---|
| Left Panel | test.png (320×240) | Quad | Slightly keystoned — drag the corners to re-align |
| Right Panel | pnggrad8rgb.png (300×300) | Quad | Mirrored keystone on the right side |
| Center Overlay | Solid color #4a9ee0 (50% alpha) | Triangle | Shows color layer blending over both panels |

Open via **File → Open** or drag the `.mmp` file onto the window.
